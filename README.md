# WatchTogether 🎬

A real-time collaborative web application that brings people together to watch YouTube videos simultaneously while chatting with each other.


## ✨ Features

- **🎥 Synchronized Video Playback**: Watch YouTube videos in perfect sync with friends
- **💬 Real-time Chat**: Built-in chat functionality for seamless communication
- **🏠 Private Rooms**: Create or join private rooms with unique IDs
- **👥 Multi-user Support**: Multiple users can join the same room
- **⚡ Real-time Updates**: Instant synchronization using WebSockets
- **🎮 Video Controls**: Play, pause, seek - all synchronized across all clients
- **📱 Responsive Design**: Works on desktop and mobile devices
- **🔒 No Authentication Required**: Jump straight into watching together

## 🛠️ Tech Stack

### Frontend
- **HTML5** - Structure and semantics
- **CSS3** - Styling and animations
- **JavaScript (ES6+)** - Interactive functionality
- **WebSockets API** - Real-time bidirectional communication

### Backend
- **Node.js** - Server-side runtime
- **Socket.io** - WebSocket library for real-time events
- **Express.js** - Web framework for Node.js

### APIs & Services
- **YouTube IFrame Player API** - Embed and control YouTube videos
- **WebSocket Protocol** - Real-time client-server communication
